# My Singing Blahaj

My Signing Blahaj is a project for the Outpost hackathon from Hack Club. It is a ikea shark plushie that uses a rotational sensor to play a different pitch of music. It then uses a program running on a laptop to show a fun ui and play the audio. THe Shark and laptop communicate over wifi and web-served files.

### BOM
- 1x ESP8266 dev board
- 1x ADXL345 breakout
- 1x usb-c cable
- 1x laptop
